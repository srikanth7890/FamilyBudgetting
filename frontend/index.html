API Endpoints

The backend API is running and ready to use. Here are the main endpoints:

Authentication

  • POST /api/auth/register/
  • POST /api/auth/login/
  • GET /api/auth/profile/

Core Features

  • GET /api/auth/families/
  • GET /api/budgets/budgets/
  • GET /api/expenses/expenses/

Getting Started

To use the full React frontend, you'll need to install Node.js and run npm install && npm start in the frontend directory. For now, you can access the admin panel at /admin/ (admin@example.com / admin123).
